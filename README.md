# 🚀 HireAI Backend

HireAI Backend is a scalable backend system for an AI-powered hiring platform that helps recruiters manage candidates, job postings, resume analysis, and AI-based candidate evaluation.

## ✨ Features

- 🔐 JWT Authentication & Authorization
- 👤 User Management
- 💼 Job Posting Management
- 📄 Resume Upload & Processing
- 🤖 AI Candidate Analysis
- 📊 Recruiter Dashboard APIs
- 📁 File Upload Support
- 🔄 RESTful API Architecture
- 🛡️ Secure Environment Configuration

## 🛠️ Tech Stack

- Python
- Django
- Django REST Framework
- PostgreSQL
- JWT Authentication
